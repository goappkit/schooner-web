Documentation

Welcome to Schooner's documentation. Find everything you need to get up and running, manage your SSH fleet, and make the most of every feature.

Where to start

New to Schooner? Begin with the Installation guide, then follow the Quick Start to connect to your first host in under two minutes.

What's in the docs

Getting Started

  • Installation — Download the DMG, drag to Applications, first launch.
  • Quick Start — Connect to your first host and learn the basic flow.

Core Features

  • Host Management — Add, edit, tag, and organize your SSH hosts.
  • Connecting to Hosts — One-click dial from the menubar, Ghostty integration, keyboard shortcuts.
  • ProxyJump Canvas — Visual topology editor for multi-hop SSH configurations.
  • File Transfer — SFTP local↔remote and server-to-server direct transfers.
  • SSH Key Manager — Generate, deploy, and audit SSH keypairs.
  • TUI & CLI — Use the sc command in any terminal.

Advanced

  • SSH Config Editing — How Schooner reads and writes ~/.ssh/config without destroying your formatting.
  • Backup & Restore — Automatic snapshots and one-click rollback.
  • Trust Manager — Audit and revoke server-to-server trust relationships.
  • Settings — All preferences explained.

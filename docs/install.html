Installation

Schooner is distributed as a standard macOS app bundle. No installer, no system password required — drag and you're done.

System Requirements

  • macOS 14 Sonoma or later
  • Apple Silicon (M1 or later) or Intel Mac
  • ~30 MB disk space

Download

Download the latest .dmg from the releases page. Open the DMG, drag Schooner.app to your Applications folder.

Gatekeeper note: Schooner is notarized by Apple. If macOS still prompts "cannot verify the developer", right-click the app and choose Open the first time.

First Launch

Schooner is a menubar app — it does not appear in the Dock. After launching, look for the ⛵ icon in your menu bar (top-right area of your screen).

On first launch Schooner will:

  1. Read your ~/.ssh/config and populate the host list automatically.
  2. Offer to install the sc CLI tool at /usr/local/bin/sc (optional but recommended).
  3. Ask for notification permission (optional — can be turned off in Settings).

Uninstalling

Quit Schooner (right-click the menubar icon → Quit), then drag Schooner.app to the Trash. User data lives at:

~/Library/Application Support/Schooner/schooner.sqlite

Delete that folder to remove all data. Schooner never writes anywhere else on your system.

Updating

Download the new DMG from the releases page and repeat the drag-to-Applications step. Your data is never touched during updates.

Settings

All Schooner preferences in one place. Open via ⛵ menubar → Settings… or ⌘,.

General

Terminal

The terminal application that Schooner opens SSH sessions in. Ghostty is selected automatically if installed. Options: Ghostty, Terminal.app, iTerm2, Alacritty, WezTerm, Warp.

Open in

Whether new SSH sessions open in a New window or a New tab in the current terminal window.

Global shortcut

The keyboard shortcut that opens the Schooner popover from anywhere. Default: ⌘ Shift S. Click the field and press your preferred shortcut to change it.

Show notification on launch

Shows a macOS notification when Schooner starts, confirming it's running in the menu bar. Useful if you auto-launch Schooner at login and want confirmation it's active. Can be turned off permanently here.

Backup retention

How many ~/.ssh/config snapshots to keep. Default: 10. Range: 1–100.

Command Line Tool

Install or reinstall the sc CLI at /usr/local/bin/sc. This creates a symlink pointing to the binary bundled inside Schooner.app.

Safety

Danger tags

A comma-separated list of tags that trigger a confirmation prompt before connecting. Default: prod, production. Hosts with these tags also show a red border on the ProxyJump canvas.

Transfer

Server-to-server mode

Global preference for how server-to-server transfers are routed:

  • Ask each time (default) — a dialog appears before each server-to-server transfer
  • Always direct — route A→B without prompting
  • Always via local — route through your Mac without prompting

About

Shows the current app version, build number, architecture, and macOS version. Useful when reporting bugs.
